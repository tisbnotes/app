Syllogisms

  • Two premises and one conclusion
  • Qualifiers such as all or some or no which tell us of the quantity being referred to.

Truth and Validity

  • An argument is valid if conclusion follows logically from the premises
  • Example:
    – All hippos eat cockraches
    – Mr porter is a hippo
    – Therefore mr porter eat cockroaches

Both premises and conclusion are false, but the argument is valid

  • Example 2:
    – All rats are teacher
    – Mr porter is a rat
    – Therefore mr porter is a teacher

Both premises are wrong, but conclusion is true. Argument is still valid

Deductive reasoning: Just becuase an argument is valid does not mean that the conclusion is true

For an argument to be true you must be able to answer yes to the following questions:

  • Are the premises true?
  • Is the argument valid?
