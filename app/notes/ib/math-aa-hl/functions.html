Functions

Basic Definitions

  • Function: A relation from a set of inputs to a set of possible outputs where each input is related to exactly one output.

    • One →\rightarrow→ One
    • Many →\rightarrow→ One

  • Domain: The set of values independent variable ‘x’ can take to give real values for ‘dependant variable y’ (ie input values)

  • Range: The set of values the dependant variable ‘y’ takes for all possible values of independent variable ‘x’ (ie output values)

Line tests

  1. Vertical line test: Check if a relation is a function (eliminates many-to-many and one-to-many relations)

  2. Horizontal line test: Check if inverse can be made from function

If a relation passes both tests, it is a one-to-one function.

    • Ex. f(x)=x3f(x) = x^3f(x)=x3

Sign Diagram

  • Find roots of given function and mark on a number line in increasing order

  • If coefficient of the highest power of xxx is positive, start with +++ to extreme right

  • If coefficient of highest power of xxx is
    negative, start with −-− to extreme right

  • If root is repeated even times, then sign does not change in alternate intervals

The root (x−1)(x-1)(x−1) is repeated even times, so sign doesn’t change in alternate intervals

Asymptotes

  1. Vertical
    • Equate denominator of rational functions to 0
  2. Horizontal
    • Analyse what happens to f(x)f(x)f(x) as x→∞x \to \inftyx→∞ and as x→−∞x \to -\inftyx→−∞
  3. Oblique:
    • A function will have an oblique asymptote if the degrees of the numerator is one greater than that of the denominator.
    • A function will never have both oblique and horizontal asymptotes.
